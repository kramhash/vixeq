# Changelog

## 0.1.0 - 2026-06-19

Initial release.

### Added

- `@viseq/core` with the UI-agnostic sequencer engine, immutable project helpers, validation, presets, smoothing helpers, and timeline utilities.
- `@viseq/react` with React hooks for driving the core engine from React apps.
- `@viseq/player-react` with an embeddable editable sequence player and bundled styles.
- Playground app with visualizer, presets, JSON import/export, and local project persistence.

### Notes

- Viseq is in early development.
- This release does not include an audio engine, MIDI support, DAW-style timeline editing UI, URL sharing, or production stability guarantees.
